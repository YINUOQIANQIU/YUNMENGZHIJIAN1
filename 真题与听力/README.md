# 听力音频文件说明

## 目录结构
- 四级听力/: 存放大学英语四级考试听力音频文件
- 六级听力/: 存放大学英语六级考试听力音频文件
- temp/: 临时文件目录

## 文件命名规范
推荐使用以下命名格式：

### 四级听力文件
- 202306-1.mp3          # 2023年6月第1套完整听力
- 202306-1-short.mp3    # 2023年6月第1套短对话
- 202306-1-long1.mp3    # 2023年6月第1套长对话1
- 202306-1-long2.mp3    # 2023年6月第1套长对话2
- 202306-1-lecture.mp3  # 2023年6月第1套讲座

### 六级听力文件  
- 202306-1.mp3          # 2023年6月第1套完整听力
- 202306-1-short.mp3    # 2023年6月第1套短对话
- 202306-1-long.mp3     # 2023年6月第1套长对话
- 202306-1-lecture1.mp3 # 2023年6月第1套讲座1
- 202306-1-lecture2.mp3 # 2023年6月第1套讲座2

## 自动导入说明
系统会自动扫描音频文件并与数据库中的试卷匹配：
1. 根据文件名中的年份、月份、试卷编号匹配试卷
2. 根据文件名中的类型分配题目
3. 自动设置音频开始和结束时间

## 支持格式
- MP3 格式音频文件
- 建议比特率: 128kbps
- 建议采样率: 44100Hz
